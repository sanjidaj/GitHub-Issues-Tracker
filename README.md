1.What is the difference between var, let, and const?
Ans: The var keyword is the older way to declare variable.It is function-scoped.Variables declared with var can be redeclared and updated. On the other hand, the let keyword is block-scoped. It can be updated but can't be redeclared in the same scope. the const keyword is also block-scoped but it can't be redeclared and updated. That means const keyword is used for values that should remain constant.

2.What is the spread operator (...)?
Ans: The spread operator expands elements of an array or object.

3.What is the difference between map(), filter(), and forEach()?
Ans: The map() method is used to transform each element of an array and returns a new array with the modified values.
However, the filter() method used to select elements based on condition and returns a new array containing the elements that satisfy the condition. Again, the forEach() is used to loop through element of an array but it doesn't return a new array.

4.What is an arrow function?
Ans: The arrow function is a shorter syntax for writing functions in JavaScript. It uses => symbol to define the function. they are commonly used for short operation.

5.What are template literals?
Ans: Template literals are a feature that is used to create string more easily using backticks(`) instead of quotation marks. With template literals, variables and expression can be added inside a string using string interpolation ${}.
